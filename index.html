Brainwave activity in your apps

MindWaveUWP makes your universal Windows apps smarter and helps them to read user brainwave data using Bluetooth EEG sensors. The library is specifically designed to be very easy, fast and deployable with minimal changes and alterations of your already working code. With only 2 extra lines of code, your application can be ready to recieve and process users brainwave data. The library can not only work with real-time EEG readings, but also provides built-in data recording for long-term data analysis.

Hardware

Library is designed for use with Bluetooth equipped MindWave Mobile EEG headset. This device is the culmination of decades of EEG biosensor technology research—all in one easy-to-control, wearable package. As the most affordable brainwave-reading EEG headset available, the MindWave and MindWave Mobile transform science fiction into reality from the comfort of your living room.

Get - set - go

First, we create a instance of MindSetConnection.

private MindSetConnection MyHeadset = new MindSetConnection()

After the MindSetConnection is created, we can directly proceed to connecting to EEG headset using the "ConnectBluetooth" method. We supply this method with the name of the bluetooth headset in string format. Please note, that this headset must be already paired with your device.

MyHeadset.ConnectBluetooth("MindWave Mobile");

That's it, Sparky! We are all set and receiving data from the headset. From now on, we can access all of the realtime reading directly from our instance of MindSetConnection just like any other integer variable:

Debug.Write(MyHeadset.RealtimeData.AlphaHigh); 
Debug.Write(MyHeadset.RealtimeData.Beta);

The result is the game controled using level of brain activity and attention of the player. This example was presented on open days of VŠB-TUO: Technical University of Ostrava.
